=== Oxy-Neon White Cursor Set ===

By: RIDDLER (http://www.rw-designer.com/user/70600)

Download: http://www.rw-designer.com/cursor-set/oxy-neon-white

Author's description:

This cursor set is a '''Oxy-Neon White''' version of the '''Oxy-Neon''' cursor family. This cursor set is completely white with transparent fills. There are 16 cursor roles in this cursor set. There are two animated cursor roles for the spinners.

Please note that this white cursor set is visible on white backgrounds but it is difficult to see for some users. To get the best viewing experience of this cursor set, use a dark background to see the cursors better. Using a black background will bring the best contrast viewing for this cursor set.
